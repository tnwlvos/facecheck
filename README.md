# FaceCheck

얼굴 인식 기반 출석 관리 시스템입니다.  
사용자 얼굴 데이터를 기반으로 출석 여부를 확인하고, 웹 환경에서 출석 정보를 관리할 수 있도록 구현한 AI 웹 서비스 프로젝트입니다.

## Project Overview

본 프로젝트는 인공지능 포트폴리오 제작 부트캠프에서 수행한 얼굴 인식 출석 시스템입니다.  
카메라를 통해 입력된 얼굴 정보를 인식하고, 등록된 사용자와 비교하여 출석 처리를 수행하는 구조를 목표로 개발하였습니다.

## Main Features

- 얼굴 인식 기반 사용자 확인
- 실시간 카메라 입력 처리
- 출석 처리 기능
- 웹 기반 사용자 인터페이스
- 출석 데이터 관리
- Frontend와 Backend 연동 구조

## Repository Structure

- my-app/
  - 웹 애플리케이션 관련 코드
  - Frontend 및 Backend 구성 파일 포함

## Tech Stack

- Python
- JavaScript
- HTML
- CSS
- Face Recognition
- Web Application
- Camera Input

## System Flow

1. 사용자가 카메라 앞에 위치
2. 시스템이 얼굴 이미지 입력
3. 얼굴 특징 추출 및 등록 데이터와 비교
4. 일치하는 사용자를 확인
5. 출석 정보 저장 및 화면에 결과 출력

## Key Implementation Points

- 얼굴 인식 모델을 활용한 사용자 식별
- 웹 기반 UI 구성
- 카메라 입력과 AI 모델 처리 연동
- 출석 정보 관리 기능 구현
- AI 모델과 웹 서비스의 통합 구조 경험

## Result

본 프로젝트를 통해 AI 모델을 단순히 학습하는 것에서 나아가, 실제 웹 서비스 형태로 연동하는 경험을 수행하였습니다.  
얼굴 인식, 사용자 관리, 출석 처리, 웹 UI를 연결하며 AI 기반 서비스 개발 흐름을 학습했습니다.
